# Tech Stack

- [TypeScript](https://en.wikipedia.org/wiki/TypeScript) is a [programming language](https://en.wikipedia.org/wiki/Programming_language). It's based on JavaScript but it has a [type system](https://en.wikipedia.org/wiki/Type_system)
- [React](<https://en.wikipedia.org/wiki/React_(software)>) is a [user interface](https://en.wikipedia.org/wiki/User_interface) [library](<https://en.wikipedia.org/wiki/Library_(computing)>) for TypeScript
- [TanStack Start](https://tanstack.com/) is a [framework](https://en.wikipedia.org/wiki/Software_framework) for React
- [shadcn/ui](https://ui.shadcn.com/docs) is a set of components for React
